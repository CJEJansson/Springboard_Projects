Springboard Data Science Career Track

Data Wrangling
Section 5.4: Collecting data from the Internet
API Mini Project

Learning Objective:
  -Practice new data wrangling techniques


Dataset: Quandl Frankfurt Stock Exchange data for
         Carl Zeiss Meditec ticker (AFX_X)

Link:    https://www.zeiss.com/meditec/int/home.html.

Assignment:
  1. Leverage the Quandl API to import the Dataset
  2. Explore the dataset and determine the json structure
  3. Complete the following tasks:
        a. Collect data from the Franfurt Stock Exchange,
           for the ticker AFX_X, for the whole year 2017.
        b. Convert the returned JSON object into a Python
           dictionary.
        c. Calculate what the highest and lowest opening
           prices were for the stock in this period.
        d. What was the largest change in any one day
           (based on High and Low price)?
        e. What was the largest change between any two days
           (based on Closing Price)?
        f. What was the average daily trading volume during
           this year?
        g. Optional- What was the median trading volume
           during this year.
